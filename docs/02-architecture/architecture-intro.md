# Architecture

One document. One source of truth.

**[Delta Labs System Architecture](./full-system-architecture)** is the single authoritative reference for Delta Labs: principles, scale targets, product scope (modules and features), logical layers, frontend, backend, database (MongoDB only), AI, infrastructure, security, observability, DR/HA, glossary, implementation roadmap, and checklist.

- **Scale, security, ops:** §7–11 and §12 (Related Documents, Runbooks placeholder).
