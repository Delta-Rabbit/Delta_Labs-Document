Delta Labs backend APIs follow **REST-style** conventions and support:

- **Authentication** (e.g. JWT or session-based).
- **Module endpoints** for courses, labs, schools, certifications, and other domains.
- **AI/chat endpoints** for BotChat and navigation.
- **Rate limiting** and consistent error responses.

Use the [full API reference](./api-reference.md) when integrating frontend with backend or when adding or changing endpoints.
